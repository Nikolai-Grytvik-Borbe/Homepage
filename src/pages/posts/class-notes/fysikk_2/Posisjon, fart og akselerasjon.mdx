### Fartsgraf kan brukes til å finne:
    * Fart ved et gitt tidspunkt
    * Tidspunktet når farten er lik en gitt verdi
    * Akselerasjonen
    * Strekningen et objekt har beveget seg i løpet av en gitt tid

### Ved bruk av python kan vi lage en fartsgraf
```python
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = 7 + (-1.4 * x)

plt.plot(x, y)
plt.xlabel('tid(s)')
plt.ylabel('fart(m/s)')
plt.show()
```
